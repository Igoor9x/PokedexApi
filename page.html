Vamos criar a sua própria Pokédex!

Primeiro, abra o seu VS Code ou o seu editor de códigos de sua preferência, monte a sua página HTML e estilize-a como quiser, utilizando o CSS ao seu gosto.

const seuElemento = document.querySelector(".suaclasseHTML");

Após isso, faremos a requisição da API. Para realizar a requisição da API, primeiro criamos uma constante com o nome "pegarItemAPI()". Toda requisição fetch (GET) é uma promessa que resolveremos com async/await, seguindo o código da seguinte forma:

const pegarItemAPI = async (dadosPokemon) => { };

Após criar a constante "pegarItemAPI", criaremos as funções para requisitar os dados da API, criando assim três novas constantes: uma para a URL (de onde obteremos os dados da API), uma para a resposta da API e outra para converter as respostas em formato JSON. Seguindo o exemplo, ficará da seguinte forma:

const pegarItemAPI = async (dadosPokemon) => {
const pegarDadosAPI = `https://pokeapi.co/api/v2/pokemon/${seuElemento}`;
const respostaAPI = await fetch(pegarDadosAPI);
const resultadoAPI = await respostaAPI.json();
return resultadoAPI;
}; (Sempre retorne o resultado da requisição da API).

Com isso, a requisição da API deve estar concluída de forma correta, mas não basta apenas obter os dados, agora precisamos mostrar as informações que estamos recebendo da API. Para fazer isso, vamos criar uma nova constante global também com "async", pois ela faz parte da requisição da API.

const mostrarItemAPI = async (dadosPokemon) => {
const dadosAPI = await pegarItemAPI(dadosPokemon);
};

Com isso, toda a parte da API está concluída, mas ainda não está pronto. Voltamos para o topo do código e agora obtemos os elementos do HTML para que possamos alterá-los com os dados da API.

const nomePokemon = document.querySelector(".nomepokemon");
const numeroPokemon = document.querySelector(".numeropokemon");
const imagemPokemon = document.querySelector(".imagempokemon");

Com os elementos HTML já obtidos no JavaScript, agora é apenas substituir os dados do HTML pelos dados da API.

nomePokemon.innerHTML = dadosAPI.name;
numeroPokemon.innerHTML = dadosAPI.id;

Após isso, estamos quase lá para ver alguma coisa. Só precisamos fazer o botão de pesquisa funcionar. Para isso, voltamos onde estão nossas constantes e obtemos a constante do botão.

const botaoProcurar = document.querySelector("meuBotao");

Depois de obter o botão HTML, criamos um evento de clique para ele, com uma função anônima (arrow function), para que algo aconteça quando ele for clicado ou pressionado. Faremos isso da seguinte forma:

botaoProcurar.addEventListener("click", (evento) => {
evento.preventDefault();
const procurarPokemon = seuElemento.value.toLowerCase();
dadosAPI(procurarPokemon);
});

let pokemonID = 0;

Após a criação da variável, faremos uma alteração na parte que altera o "innerHTML". Abaixo das constantes de "innerHTML", adicionamos o seguinte código:

pokemonID = dadosAPI.id;

Após isso, obteremos os botões nas constantes, da seguinte forma:

const botaoProximo = document.querySelector(".proximo");
const botaoAnterior = document.querySelector(".anterior");

E, por fim, criamos os eventos para cada botão:

botaoProximo.addEventListener("click", () => {
pokemonID += 1;
dadosAPI(pokemonID);
});
botaoAnterior.addEventListener("click", () => {
pokemonID -= 1;
dadosAPI(pokemonID);
});

E, por fim, chamamos a função principal para iniciar:

dadosAPI(pokemonID);

Com isso, finalizamos toda a API e ela deve funcionar normalmente, tanto pesquisando pelo número quanto pelo nome do Pokémon.
